## General Overview for Lucky Llama
Lucky LLama is a web app with the sole purpose of providing interactive ways for randomizing and making party games interactive. It has four gamemodes: Rolling Roullete which is a spinner that holds multiple text, Capricious Cards where mutiple cards are set and is flipped on its backside selecting one unflips the card, Blithe Bottle is just a spinning bottle kinda like a digital spin the bottle, and spinning slots which is a slot machine. We will be working on the Rolling Roullete first.

## Rolling Roullete
When the gamemode card for rolling roullete is clicked, it will redirect to a new page where it displays the wheel. There is a floating icon at the bottom that when clicked we can input the names to be listed in the wheel. Similar to the chat Icon here: https://elijah-alonzo.vercel.app/.
